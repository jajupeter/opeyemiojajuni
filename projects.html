DeepRoute: RL + SDN for Real-Time Traffic Rerouting

Scope

Manage WAN traffic spikes by learning optimal paths and rerouting flows in real time using reinforcement learning on SDN.

Work Done

  • Implemented UCB/Deep RL to learn path selection and rerouting strategies.
  • Demonstrated on Mininet and Chameleon Testbed (BYOC) with OpenFlow switches.
  • Measured throughput improvements and flow diversions under traffic variability.

Impact

  • Improved path utilization and reduced congestion for elephant/mice flows.
  • Actionable for production SDN environments needing adaptive control.

DynamicDeepFlow: Unsupervised Clustering for Flow Changes

Scope

Identify evolving network traffic patterns and anomalies in ESnet using VAE feature compression + K‑means++ clustering.

Work Done

  • Designed VAE to compress flow features; clustered with K‑means++.
  • Evaluated on real ESnet traffic to separate normal vs anomalous patterns.

Impact

  • Real‑time operational insights for monitoring big vs small flows.
  • Supports proactive mitigation of congestion and anomalies.
